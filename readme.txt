Backup Rocks is a unique and most reliable automatic real time backup service for your WordPress site. Backup Rocks provide both free and paid subscriptions.

== Description ==
Backup Rocks takes automatic real time backup of your wordpress site with easy restore option and option to migrate to new host.

Top Features:

* Automatic Real-time WordPress Backup
* Complete WordPress Backup - files backup and database backup
* Incremental backups
* All Time Backup History
* Easy restore and migration option from backup
* Managed offsite backup
* Incremental backups
* Encrypted and Secure Backups
* 24/7 hours support from WordPress experts
* Easy to use dashboard to manage all backups
* Free subscriptions available

== Installation ==

= Automatic installation =

* To do an automatic install of Backup Rocks plugin, log in to your WordPress dashboard, navigate to the Plugins menu and click Add New.
* In the search field type Backup Rocks and click Search Plugins. Once you've found our WordPress backup plugin you can install it by simply clicking "Install Now".
 After installation, click Activate Plugin. Now to register with Backup Rocks go to https://backup.rocks and then click on sign up. pop up will appear which will ask you for the email password and site url. Please use valid email and strong password, for the site url please use the url of the site that you want to backup. or url of the site that you have installed backup plugin on. URL must be complete for example:-
if your site is example.com then please enter the site url as http://example.com.
if your site is www.example.com  then please use http://www.example.com.
if your site is ussing ssl then use https://example.com (with or without www depending on your site url).
after filling this information click on Register. then a new pop up will appear which will ask you to select a plan. After selecting the plan of your choice (Free or premium). It provide you with the license key and a link to download the plugin. (you don't need to download it again. As you have already done so in above steps.). Copy the license key and then go to your website admin panel then click on backup.rocks menu. paste your key and then click on save. After this you will see a save button changed to Activate license. click on Activate License. that's all :). Now after few minutes. you can login to backup.rocks dashboard to see your backup.rocks in action.


= Manual installation =

The Backup Rocks plugin can also be installed manually by downloading it from the WordPress plugin repository and then uploading it to your web server via your favorite FTP application. The WordPress codex contains [instructions on how to do this here](http://codex.wordpress.org/Managing_Plugins#Manual_Plugin_Installation). Once completed, follow the same instructions as above to register with us and obtain a license key. Your backup should be initiated immediately.
